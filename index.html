EduMind: AI-Powered Study Companion

Python, Flask, React Native, LLMs

Project Overview

EduMind is an AI-driven learning platform designed to enhance the educational experience through real-time explanations, quiz generation, and lecture summarization.

Key Features

  • AI-powered chat interface using the Mistral model for personalized learning assistance
  • Automatic quiz generation based on uploaded study materials
  • Lecture summarization tool to condense complex information
  • Optimized data retrieval using FAISS/ChromaDB for fast response times
  • Offline capability for learning without internet connectivity

Technical Implementation

The application was built with a Python Flask backend that handles the AI processing and data management. The frontend was developed using React Native for cross-platform compatibility. We integrated self-hosted LLMs to ensure data privacy and reduce dependency on external APIs.

Unisphere: College Event Management App

React Native, Node.js, MongoDB

Project Overview

Unisphere is a comprehensive mobile application designed to streamline the discovery, registration, and management of college events, providing students with a centralized platform for all campus activities.

Key Features

  • Cross-platform mobile app built with React Native and Expo
  • Real-time event updates and notifications
  • Interactive UI with animated event cards and image galleries
  • User profile management with event history
  • Event registration and attendance tracking

Technical Implementation

The application uses React Native for the frontend with React Native Reanimated for smooth animations. The backend is built on Node.js with Express, and MongoDB serves as the database. Real-time updates are implemented using Socket.io for instant notifications about event changes.

Image Segmentation and Object Detection System

Python, YOLOv3, Pointrend

Project Overview

This project focuses on developing an advanced system for image segmentation and object detection, capable of precisely identifying and delineating multiple objects within complex visual environments.

Key Features

  • Integration of YOLOv3 for fast and accurate object detection
  • Implementation of Pointrend for precise boundary delineation
  • Multi-object detection in complex image environments
  • Optimized performance for real-time processing
  • Customizable detection parameters for different use cases

Technical Implementation

The system was implemented in Python using OpenCV and PyTorch. YOLOv3 was used as the base detection model due to its speed and accuracy, while Pointrend was integrated to improve the precision of object boundaries. The system was optimized for performance to enable real-time processing on standard hardware.
